Conceptos básicos de HTML

¿Qué es HTML?

HTML es un lenguaje de etiquetas de hipertexto (HyperText Markup Language) y es usado para el desarrollo de paginas web.
Los lenguajes de marcado suelen confundirse con lenguajes de programación. Sin embargo, no son lo mismo, ya que el lenguaje de marcado no tiene funciones aritméticas o variables, como poseen los lenguajes de programación.

HTML5 es un estandar que sirve de referencia para la elaboración de la estructura y contenido de una página web, que puede tener texto, imágenes, tablas, videos, juegos, entre otros.

World Wide Web Consortium (W3C) o Consorcio WWW, es la organización dedicada a la estandarización de casi todas las tecnologías ligadas a la web, sobre todo en lo referente a su escritura e interpretación.

Un lenguaje de etiquetado/marcado es una forma de codificar un documento que, junto con el texto, etiquetas o marcas que contienen información adicional acerca de la estructura del texto o su presentación.

¿Qué es una etiqueta?

Una etiqueta es una palabra clave con multiples atributos la cual HTML identifica para realizar una función en el documento web.

Existen muchos tipos de etiquetas, cada una tiene su función, ya sea para modificar el nombre de la página, los tipos de letras, darle estructura a la página o mostrar información al usuario.
